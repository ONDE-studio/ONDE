# Бизнес-процесс Заявки на Заказ (Order Flow)

В магазине ONDE **полностью отсутствует онлайн-оплата непосредственно на сайте**.

## Этапы Заявки:

1. **Просмотр и Корзина**:
   Покупатель выбирает изделия в каталоге или открывает карточку товара `/products/[slug]`, указывает количество и добавляет в корзину.

2. **Оформление Заявки (`/checkout`)**:
   - Ввод контактных данных и точного адреса.
   - Запрос и выбор реального варианта доставки по API (СДЭК, Почта России, Яндекс) или варианта со сделкой в Telegram.
   - Подтверждение Пользовательского Соглашения и Согласия на обработку данных.

3. **Регистрация Заявки (`/api/orders/create`)**:
   - Сервер пересчитывает стоимость товаров из базы данных (без доверия ценам браузера).
   - Генерирует уникальный публичный номер заявки вида `ONDE-YYYYMMDD-XXXX`.
   - Сохраняет транзакционную запись в таблице `orders`.
   - Отправляет мгновенное серверное уведомление в Telegram администраторам через Telegram Bot API.

4. **Экран Подтверждения (`/orders/[id]?token=...`)**:
   - Отображает публичный номер, состав и детали.
   - Предоставляет кнопку «Скопировать детали заказа» (с адаптивным фоллбеком для буфера обмена).
   - Предоставляет кнопку «Перейти к согласованию в Telegram» с предустановленным сообщением.

5. **Персональное Согласование**:
   В Telegram-чате клиент и менеджер студии ONDE окончательно подтверждают наличие, цвет, сроки печати и способ оплаты.
